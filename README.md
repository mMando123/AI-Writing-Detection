# AI-Writing-Detection Bypass System (Version 1)

هذا النظام هو محرك متقدم مبني بلغة بايثون (Flask) لتجاوز تقنيات كشف المحتوى المكتوب بالذكاء الاصطناعي (AI Detection Bypass).

## الميزات
- **التحليل العلمي**: يقيس الحيرة (Perplexity) والانفجارية (Burstiness) باستخدام النماذج الإحصائية مثل N-gram Entropy ومؤشر Gini.
- **محرك التجاوز**: يقوم بتطبيق 6 تقنيات لتجاوز أنظمة الكشف (إزالة البصمات، أنسنة الأسلوب، رفع الانفجارية، وغيرها).
- **المعالجة المتكررة (Convergence)**: يقوم بمعالجة النص بشكل متكرر حتى يصل إلى درجة "بشرية" آمنة.
- **واجهة عصرية**: واجهة مستخدم مبنية بـ Glassmorphism تدعم اللغة العربية.

## التشغيل
```bash
pip install -r requirements.txt
python app.py
```
الخادم سيعمل على `http://localhost:5000`
